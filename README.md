# MacOS文件管理器
语言：[简体中文](./README.md) | [English](./README.en.md)

一个简洁高效的本地文件管理器，支持双栏浏览、终端快捷打开、磁盘信息查看与右击菜单操作，并提供文件预览能力。

## 功能特性

- 双栏浏览：同时查看与操作两个目录
- 终端快捷打开：可通过右击或顶部功能菜单打开终端
- 磁盘信息：查看磁盘与分区信息
- 右击菜单：常用文件操作入口
- 文件预览：常见文件类型的快速预览

## 键盘操作

- Shift + 左击：多选
- Command + 左击：多选
- Command + A：全选
- F5：刷新
- Delete：删除

## 使用提示

- 双栏便于对比并在目录间进行操作
- 右击菜单与顶部功能菜单提供常用操作入口
- 界面未更新时可使用 F5 刷新
